In-Person Group Classes

In January and February, we offer week-end enrichment classes of up to 4 students. Payment is required to reserve spot, no walk-ins accepted.

Sign Up: nora@blueridgeboost.com

Where: 1629 Appian Way, Charlottesville, Virginia (Pantops area).
Conditions to attend: proof of COVID-19 vaccination, commitment to wear a mask for the duration of the camp.

Coding (grades 1-3)

When: Saturday 10am - 11am, January 22 - February 26
What: Each student follows a selection of instructor lessons, Scratch projects, Lego WeDo projects, and Coffee Script programming exercises.
  1. Algorithmic thinking (20 min). Students learn the basics of a computer program and how a program is executed through play.
  2. Block-language programming (20 min). Students solve instructor-selected exercises from Code Monkey, the Beaver Achiever course, on provided Chromebooks.
  3. Choice time (20 min). students may choose their activity from coding in Code Monkey, Lego robots building and programming using WeDo Lego sets, Makey Makey, mBot, and Cubetto.
  4. Work at home. Students are able to use the programming interface from home to practice.
Who: Any student interested in coding (prior experience not required).

Cost: $250 for six 1-hour sessions.

Coding (grades 2-4)

When: Saturday 11:30am - 12:30pm, January 22 - February 26
What: Coffee Script (an easy-to-learn first programming language that runs in your web browser) and Python (a powerful and widely used programming language)
  1. Programming Constructs (30 min). Students basic programming constructs including conditionals, loops, and sorting algorithms.
  2. Programming (30 min). Students solve instructor-selected exercises from Code Monkey, Coding Adventure or Banana Tales courses, on provided Chromebooks.
  3. Work at home. Students are able to use the programming interface from home to practice.
Who: Any student interested in coding.

Cost: $250 for six 1-hour sessions.

Advanced Coding (grades 4-8)

When: Saturday 12pm - 2pm, January 22 - February 26
What: Python (a powerful and widely used programming language) and algorithms and problems from USA Computing Olympiad Bronze Level
  1. Algorithms (25 min). Students learn basic algorithms including sorting, and graph algorithms.
  2. Break (5 min).
  3. Programming Practice (25 min). Students solve instructor-selected exercises from Code Monkey, Banana Tales or Coding Chatbots courses, on provided Chromebooks.
  4. Break (10 min).
  5. Problem Solving (55 min). Students work in pairs to solve USA Computing Olympiad Problems.
  6. Work at home. Students are able to use the programming interface from home to practice.
Who: Any student interested in coding with some prior programming experience.

Cost: $500 for six 2-hour sessions.

Problem Solving for Fourth and Fifth Graders

When: Sunday 11am - 12pm, January 23 - February 27
What: Problem solving strategies for elementary school math.
The problems are selected from a variety of contests and books, including Math Olympiads for Elementary and Middle Schools, Math Circle by the Bay: Topics for Grades 1-5, and A Moscow Math Circle: Week-by-Week Problem Sets.
Who: Students at fourth or fifth grade-level math.

Cost: $200 for six 1-hour sessions.

Pre-algebra

When: Sunday 12:30pm - 2pm, January 23 - February 27
What: Pre-algebra. Students learn a rigourous approach to pre-algebra using the Art of Problem Solving, Prealgebra text, supplemented with Desmos activites.
Who: Students at or above fifth grade level math.

Cost: $300 for six sessions.
